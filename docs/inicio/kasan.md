# Instalar Kasam
- Para instalar
```console
sudo add-apt-repository ppa:kazam-team/stable-series
sudo apt-get update
sudo apt-get install kazam
```
- Ajustar para que no aparesca negro
```console
sudo nano/etc/gdm3/custom.conf
```
- Colocar en false
```console
WaylandEnable=false
```
